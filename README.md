# Nisrine Mazouzi Portfolio

A modern, professional portfolio website showcasing software engineering and cybersecurity expertise.

## 🌟 Features

- **Modern Dark Theme** - Sleek design with cyan/purple accents
- **Fully Responsive** - Works on desktop, tablet, and mobile
- **Smooth Animations** - Scroll reveal effects and transitions
- **Interactive Elements** - Typing effect, hover effects, and more
- **Professional Sections** - About, Education, Experience, Projects, Skills, Certifications, Contact

## 🚀 Quick Start

1. Open `index.html` in your browser to preview locally

## 📦 Deployment to GitHub Pages

To get a shareable link for your interviews, follow these steps:

### Step 1: Create a GitHub Repository
1. Go to [GitHub](https://github.com)
2. Click the **+** icon → **New repository**
3. Name: `portfolio`
4. Make it **Public**
5. Click **Create repository**

### Step 2: Upload Files
```bash
# Navigate to your portfolio folder
cd /Users/nisrinemazouzi/untitled folder 3/portfolio

# Initialize git
git init

# Add all files
git add .

# Commit
git commit -m "Initial portfolio website"

# Add your GitHub repository
git remote add origin https://github.com/nisrine-m/portfolio.git

# Push to GitHub
git push -u origin main
```

### Step 3: Enable GitHub Pages
1. Go to your repository on GitHub
2. Click **Settings** → **Pages** (in left sidebar)
3. Under **Source**, select **main** branch
4. Click **Save**
5. Wait 1-2 minutes, then your site will be live at:
   `https://nisrine-m.github.io/portfolio`
